# PokeAtlas

### 🚀 Live Demo: [https://pokeatlasrepo.vercel.app](https://pokeatlasrepo.vercel.app)

PokeAtlas is a modern, responsive, and feature-rich Pokémon encyclopedia (Pokédex) built using Next.js, Tailwind CSS, and Framer Motion. It leverages the PokeAPI to fetch live data about Pokémon, abilities, moves, regions, and types, offering an immersive interface with interactive statistics charts, type matchup guides, and smooth page transitions.

---

## Features

- **Pokedex**: Browse through the list of Pokémon, search by name or ID, and filter by types or generations.
- **Detailed Pokémon Pages**: View comprehensive stats, abilities, move sets, evolution chains, and base stat charts.
- **Abilities & Moves**: Look up details about Pokémon moves and abilities.
- **Regions**: Explore different regions of the Pokémon world and their corresponding generation's Pokémon.
- **Type Matchups**: View and calculate type advantages and weaknesses.
- **Comparison Tool**: Compare base stats of different Pokémon side-by-side.
- **Responsive Design**: Sleek UI supporting both dark and light modes.

## Tech Stack

- **Framework**: Next.js 15
- **Styling**: Tailwind CSS, CSS Variables
- **Animations**: Framer Motion
- **Icons**: React Icons
- **Data Source**: PokeAPI

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploying to Vercel

The project is configured for easy deployment on [Vercel](https://vercel.com). Simply run:

```bash
vercel
```
